DCT based speech enhancement in vehicular environments

Developed a speech enhancement system using modified DCT to combat the effects of noise in vehicular environments. Project focuses of implementing an alternative for the traditional spectral subtraction method that uses FFT by exploiting the properties of discrete cosine transform (DCT).

Popcorn saver- Movie search and recommender system

Popcorn Saver is a movie recommendation web application developed as a project for 2IMW15: Web Information Retrieval and Data Mining, during the first quartile of the 2015/2016 academic year at TU Eindhoven. The search feature includes boolean and parametric search. User based, item based and SVD recommender are used to provide intelligent recommendations to the users. Other features such as tweet mood detection, query suggestion and correction are implemented to enhance user experience.

Internet Usage - Data Visualization

Web application developed as a project for the course 'Visualization' during the second quartile at TU Eindhoven. Three techniques are implemented using D3.js to visualize the dataset, which contains the percentage of internet connected inhabitants in different countries for the years 1990 to 2011. The applications consists of a Choropleth map, time-series line chart and a vertical bar chart. The three types of visualizations are linked interactively to gain insights into the development of internet in different countries.

Cryptocurrency Analysis

Built a Bitcoin database on PostgreSQL using the bitcoinj Java library. The bitcoin data was converted to a graph with the nodes representing transactions and edges as timestamps to perform temporal analysis using Apache Flink and Gelly.

Historical Graph Analysis

Developed a Temporal Graph Index for handling large temporal graph data. This system allows indexing of temporal data and analyze dynamic temporal analysis. It includes a testing framework that evaluates the Index and Query manager. This project was developed for the course Database Technology(2IMW20).

Geography Information System

This web application was developed as a project for the course Adaptive Web-based Systems (2IMW25). A popular application area of Adaptive Web-Based Systems research is the use of adaptation in on-line information systems. The system that we designed and implemented is an online encyclopedia about geography of the world. It includes adaptations such as language based on browser settings, show locations based on GPS, different layouts, visual and verbal learning styles.
